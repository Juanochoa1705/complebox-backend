# Objetivos del proyecto Complebox

## Objetivo general
Desarrollar una plataforma integral (web y móvil) que automatice la gestión de correspondencia y paquetes en conjuntos residenciales, mejorando la trazabilidad y la comunicación entre residentes, vigilantes y mensajeros.

## Objetivos específicos
1. Implementar un sistema centralizado de registro de pedidos con control de estado.  
2. Desarrollar una API REST para la comunicación entre la base de datos y la aplicación móvil.  
3. Crear interfaces intuitivas para cada tipo de usuario (administrador, vigilante, residente, mensajero).  
4. Generar notificaciones automáticas sobre nuevos pedidos, entregas y confirmaciones.  
5. Permitir a las empresas de seguridad y mensajería integrarse fácilmente al sistema.
